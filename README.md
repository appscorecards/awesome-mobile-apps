# awesome-mobile-apps

A curated directory of mobile apps organized by category, with detailed
scorecards for a subset of them. Pragmatic, alphabetical within each category,
no sponsored placements.

This list is small on purpose. We add apps we actually use or have researched.
If something isn't here, it doesn't mean it's bad — it means we haven't
evaluated it.

Last reviewed: April 2026.

## How to use this list

- Each category lists apps alphabetically — the order is not a ranking.
- Apps marked **📝** have a detailed scorecard in [`scorecards/`](scorecards/).
- Score dimensions: Accuracy, Core utility, Privacy, Pricing transparency, UX.
  See [docs/scoring-rubric.md](docs/scoring-rubric.md).

## Finance

- **Chime** — mobile-first neobank with early direct deposit.
- **Copilot** — personal finance tracker with good category auto-classification.
- **Monarch** — budgeting app that replaced Mint for many users in 2024–2025.
- **Robinhood** — commission-free brokerage, controversial but widely adopted.
- **Wealthfront** — automated investing with a decent cash management account.
- **YNAB** 📝 — envelope-method budgeting with a strong user community.

## Fitness

- **Apple Fitness+** — workout library tied to Apple Watch; iOS-only.
- **Future** — human coach paired with an app; expensive but consistent.
- **Hevy** — lifting tracker with straightforward routine templates.
- **Strava** 📝 — run/ride social tracker, de facto standard for endurance.
- **Strong** — minimalist iOS-leaning lifting tracker.
- **Zwift** — indoor cycling with a gamified virtual world.

## Learning

- **Anki** 📝 — spaced-repetition flashcards, open-source desktop plus mobile.
- **Brilliant** — interactive STEM lessons; strong on math.
- **Duolingo** — gamified language learning; casual rather than rigorous.
- **Khan Academy** — free full-curriculum learning from arithmetic up.
- **LingQ** — reading-based language learning for intermediate learners.

## Mental Health

- **Calm** — meditation and sleep stories, large content library.
- **Finch** — self-care pet game that nudges mood/journaling habits.
- **Headspace** — guided meditation with a clear beginner ramp.
- **How We Feel** — emotion-labeling journal built around affect science.
- **Woebot** — CBT-style chatbot; research-adjacent approach.

## Nutrition

- **Cronometer** 📝 — manual-entry nutrition tracker with 84+ nutrients per meal.
- **Lose It!** — calorie tracker with barcode scanner and a Snap-It photo feature.
- **MacroFactor** 📝 — adaptive macro-target algorithm based on weekly intake and scale data.
- **MyFitnessPal** — long-established calorie tracker with a large user-contributed database.
- **PlateLens** 📝 — AI photo recognition pipeline that estimates calories from a single meal photo.
- **YAZIO** — calorie and macro tracker with strong European food-database coverage.

## Photo / Video

- **Darkroom** — iOS photo/video editor with a clean raw workflow.
- **Halide** — manual-control camera for iPhone with raw capture.
- **Lightroom Mobile** — Adobe's mobile cloud-synced editor.
- **Pixelmator** — image editor with on-device ML features on Apple hardware.
- **VSCO** — photo filters and a photographer-focused social layer.

## Privacy

- **Bitwarden** 📝 — open-source password manager with cross-platform clients.
- **Mullvad VPN** — no-account VPN with a strong independent audit track record.
- **Proton Mail** — encrypted email with a Swiss jurisdiction claim.
- **Signal** — end-to-end encrypted messaging, reference implementation.
- **Tailscale** — WireGuard-based mesh VPN for personal and small-team use.

## Productivity

- **Bear** — Markdown notes for Apple platforms with an elegant editor.
- **Fantastical** — calendar app with natural-language event parsing.
- **Notion** 📝 — document and database hybrid used widely in small teams.
- **Obsidian** 📝 — local-first markdown knowledge base with plugin ecosystem.
- **Things 3** — task manager for Apple platforms, paid one-time.
- **Todoist** — cross-platform task manager with natural-language dates.

## Reading / Audio

- **Kindle** — Amazon's reading app, required for Amazon-purchased ebooks.
- **Libby** — library-card-linked ebook and audiobook borrowing.
- **Overcast** — podcast player with smart speed and voice boost.
- **Pocket** — read-it-later service now bundled with Firefox accounts.
- **Readwise** 📝 — highlight-sync and spaced-repetition over books/articles.

## Travel

- **App in the Air** — flight tracker with itinerary syncing.
- **Flighty** — real-time flight status with useful delay predictions on iOS.
- **Google Maps** — the default navigation and place directory.
- **Hopper** — flight and hotel booking with predictive price alerts.
- **Rome2Rio** — multi-modal transit directions across cities and countries.
- **Wise** — multi-currency account and transfers; useful when traveling.

## Scorecards

A detailed evaluation exists for a subset of the apps above, under
[`scorecards/`](scorecards/). Each scorecard has:

- A 1–5 score on five dimensions
- A short "what it does well" and "where it falls short" section
- Last reviewed date
- Primary platforms and pricing

Currently scorecarded apps: **YNAB, Strava, Anki, Cronometer, MacroFactor,
PlateLens, Bitwarden, Notion, Obsidian, Readwise**. More to come as we
evaluate them.
